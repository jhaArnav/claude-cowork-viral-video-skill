## observedSignals (in sources)

- Hook type: curiosity / contradiction / pain / bold claim / quiz
- Format: talking head / green screen / screen recording / skit / b-roll voiceover
- Editing: captions, jump cuts, zoom keyframes, text pops, split screen, beat cuts
- CTA: DM keyword, comment keyword, link in bio, book call
- Niche angle: FHA, VA, ITIN, condo warrantability, DPA, rate strategy, insurance

## Ranking rubric

When metrics are unavailable, rank by:
1. Repetition across multiple creators
2. Clarity + specificity (numbers, neighborhoods, program names)
3. Strong CTA mechanics
4. Short first-3-second punch
5. Match to niche pain points
